# Dataset Card: PathMNIST

## Project

MedImage DataOps

## Dataset

PathMNIST from MedMNIST.

## Purpose

This dataset is used to demonstrate medical imaging data management for AI workflows.

## Modality

Histopathology image classification.

## Task Type

Multi-class image classification.

## Data Management Workflow

This project includes:

- dataset download
- metadata extraction
- image dimension validation
- label validation
- class distribution analysis
- train/validation/test split inspection
- dataset quality reporting
- sample visualization
